# SIGNUP UI DESIGN SPEC

## GLOBAL RULES

- This UI is FINAL
- DO NOT redesign anything
- DO NOT change spacing, alignment, or structure
- DO NOT replace components
- ONLY use existing components

---

## COMPONENTS

- InputField
- PrimaryButton
- TrustCard
- AvatarBadge
- LogoHeader

---

## LAYOUT ORDER (STRICT)

1. Background (map with opacity)
2. Pins (red + blue, subtle)
3. LogoHeader (centered, large emphasis)
4. AvatarBadge (centered)
5. Title section (centered)
6. Form (left aligned)
   - Full Name
   - Phone Number (+63 prefix)
   - Email (optional)
7. PrimaryButton (Continue)
8. Helper text (OTP notice)
9. Divider (OR)
10. Sign In text
11. TrustCard (centered)

---

## VISUAL RULES

- Logo must be dominant
- Inputs must be readable (clear text contrast)
- Background must be subtle (low opacity)
- Pins must NOT distract
- Button must be strongest color (red)

---

## INPUT RULES

Each InputField must have:
- Label (uppercase)
- Input text (clear and readable)
- Optional helper text

---

## BUTTON RULE

- Text: "Continue"
- No icon
- Full width
- Strong red color

---

## TRUST CARD RULE

- Center aligned
- Icon on top
- Text centered
